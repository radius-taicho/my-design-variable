# _Variables（自分のデザイン変数）

> ここが my-design-variable の核心。
> 理論を学んで、実務で試して、自分だけの変数を育てる場所。

---

## 2種類のノート

### 🟡 [[hypothesis-variables]]（仮説変数）
まだ不確かだけど、引っかかっているもの。
「これが自分の変数かもしれない」という仮説の置き場。
思いついたらすぐ書く。精度は問わない。

### 🟢 [[my-variables]]（自分の変数）
実務と理論の両方で繰り返し確認できた、自分の軸。
複数案件を経て「これは間違いない」と思えるもの。

---

## 変数の育て方

```
直感・気づき
    ↓
hypothesis-variables に書く
    ↓
実務で試す・理論と照合する
    ↓
確信に変わったら confirmed-variables へ移動
```
