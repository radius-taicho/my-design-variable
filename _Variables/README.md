# _Variables

定数を集め、変数を育て、応用力を手に入れる。

---

## 2種類のノート

### 🟡 [[hypothesis-variables]]（仮説の変数）
日常や実務で気づいたことを言語化する置き場。
精度は問わない、思いついたらすぐ書く。

### 🟢 [[my-variables]]（自分の変数）
実務と理論の両方で繰り返し確認できた、自分の軸。

---

## 変数の育て方

```
日常の気づき・実務の経験（＝定数）
            ↓
  hypothesis-variables に書く
            ↓
  実務で試す・理論と照合する
            ↓
    my-variables へ移す
```
